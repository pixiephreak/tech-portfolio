Playground Adventures

A mobile-first web app to help parents find accessible playgrounds while traveling or at home. Continuing work on the project created and developed by Meagan Gallagher, Cyrus Vossoughi, Alfred Avor, and Jane Claire Remick. Designed architecture and wrote JS code for integration of Google Maps/Places APIs and NPR Playgrounds for Everyone dataset to search/retrieve/display accessible playgrounds. Open-sourced the project with hopes to integrate more area-specific civic data sets

Friend Finder

Friend Finder is Node.js app which uses routes and a custom api to store and match users based on similar or differing political beliefs.

Wing

Wing. A location-based Crxxgslist on Demand. Post a task and declare your terms. Take a task and get compenstaed directly. No corporations. No 1099s. The beta version is still in development, but shows what I've been working on to combine client and server js, create routes, and build custom apis.

For this project, I handled sending and retrieving form data between the client and server, using Sequelize, Express, Ajax requests and vanilla JavaScript. I also handled the interactive map component by using mapbox-gl.js and the mapbox-gl-geocoder.

GreenMyScreen is an iOS app (available for download in the App Store) that allows users to replicate the aesthetics of commercial stock photography by filling the viewing area of their Apple device with a solid green field. Using a “green screen” is also known as “chromakeying,” a technology which allows an editor to insert any content of her choice during film/video postproduction. On websites like iStockPhoto.com by Getty Images, advertisers and designers can can find images of mobile devices in which the screen area of the device is already prepared with a green rectangle. GreenMyScreen allows amateur and professional users alike the opportunity to experience this visual phenomenon IRL and create marketable images. Actual chromakeying results my vary.
