Playground Adventures 2.0

Product

Playground Adventures is a mobile-first web app to help parents find accessible playgrounds while traveling or at home. This version continues work on the beta project created and developed by Meagan Gallagher, Cyrus Vossoughi, Alfred Avor, and Jane Claire Remick.

Contributions

I designed the Firebase-Google Maps API architecture and wrote JS code to search/retrieve/display accessible playgrounds. I also redesigned and styled the UI for version 2.0 and am open-sourcing the project with hopes to integrate more area-specific civic data sets. Furthermore, for version 2.0, I used an MVVM pattern and data-binding implemented with the Knockout JavaScript library.

(beta in development)

Product:

Wing. A location-based Crxxgslist on Demand. Post a task and declare your terms. Take a task and get compensated directly. No corporations. No 1099s.

The beta version was created and developed by Meagan Gallagher, Robert Chambliss, Olugbenga Akingbe, and Jane Claire Remick. Features such as validation are still in development by the team in order to create a working app.

Contributions:

For this Node.js app, I handled sending and retrieving form data between the client, server, and MySQL database using Sequelize, Express, Ajax requests and vanilla JavaScript. I also handled the interactive map component using mapbox-gl.js and the mapbox-gl-geocoder.

GreenMyScreen

Product:

GreenMyScreen is an iOS app (available for download in the App Store) that allows users to replicate the aesthetics of commercial stock photography by filling the viewing area of their Apple device with a solid green field. Using a “green screen” is also known as “chromakeying,” a technology which allows an editor to insert any content of her choice during film/video postproduction. On websites like iStockPhoto.com by Getty Images, advertisers and designers can can find images of mobile devices in which the screen area of the device is already prepared with a green rectangle. GreenMyScreen allows amateur and professional users alike the opportunity to experience this visual phenomenon IRL and create marketable images. Actual chromakeying results my vary.

Approach:

In GreenMyScreen, I’m thinking about the green screen as a visual symbol of infinite possibility, but also as a site of intrusion. Due to developments in UI and UX design, the content on the device screen is less and less subject to user control. I built GreenMyScreen in Swift4 using Xcode and deployed to the App store through the Apple Developer Program. It was show at code/switch, an exhibition Juried by Kayla Anderson for the Woman Made Gallery in Chicago.
